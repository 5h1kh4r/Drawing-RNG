# Data Handling

Volunteers should be told:

- Do not draw names.
- Do not draw signatures.
- Do not draw initials.
- Do not draw passwords.
- Do not draw private symbols.
- Draw naturally; do not trace perfectly.

Recommended dataset versions:

```text
raw collection            everything saved
clean prompted doodles    manually filtered prompt dataset
clean enrollment          true 3-attempt seed enrollment dataset
excluded                  discarded samples + reasons
```

Never delete raw data during active research unless you have a backup. Use separate clean tables instead.
