Interactive research demo

Can a drawing become a repeatable seed?

Draw one remembered symbol three times. Drawing-RNG measures how consistently you reproduce it and generates deterministic demo outputs.

Prototype only, not production security

What you will do

The full demo takes a few minutes and stores no image file. It works from captured pen or pointer strokes.

Draw 3 times Analyze Review output Test redraw
Start the drawing demo

Before you begin

Use a made-up symbol you can redraw. Do not draw a signature, password, private symbol, name, or other identifying information.

This project studies usability and consistency. It does not claim that drawings are stronger than random cryptographic seeds.
